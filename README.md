## Getting Started

This repository is set up as a flexible personal project space. You can add your own code, documentation, and experiments here.

## Usage

Clone the repository and start adding your own projects:

```bash
git clone https://github.com/Manuram07/RAM-IPA.git
cd RAM-IPA
```

Try the example code:

```bash
# Run the example Python script
python3 src/example.py

# Run the example tests
python3 tests/test_example.py
```
